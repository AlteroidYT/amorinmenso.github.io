✨ ¿Qué es esto?

*AmorInmenso* es un regalo digital, una carta interactiva hecha a mano con HTML, CSS y mucho corazón.

No es una página más: es una forma de decir “te amo” en cada detalle.

🛠️ ¿Qué contiene?
